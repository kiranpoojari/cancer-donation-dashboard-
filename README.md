# Cancer Donation Tracking Dashboard

This is a simple static dashboard built for an internship assignment.  
It displays dummy donation data including:

- Total donation amount
- Number of donors
- Chart showing donation trend
- Table of recent donations

The page is responsive and built using HTML, CSS, Bootstrap, and Chart.js.
